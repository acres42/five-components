## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/five-components.git
cd five-components
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

Then open your browser to:

```
http://localhost:3000
```

---
